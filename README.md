# -Windows-Server-PKI-Lab-Autoridade-Certificadora-com-AD-CS
Este repositório documenta a configuração de uma Autoridade Certificadora (CA) local em ambiente de laboratório, cobrindo desde a instalação do AD CS até a emissão automática de certificados digitais para clientes da rede via GPO. Entender como funciona uma PKI corporativa na prática.
